# WEB-MAP Navigation System

A web-based interactive navigation system built using **Leaflet.js**, **CARTO basemaps**, and **OpenStreetMap (OSM)** services.  
The application allows users to compute routes between locations, use their current geolocation, and visualize navigation paths dynamically on a web map.

## Features
- Interactive map visualization using Leaflet
- CARTO Voyager basemap integration
- Address-based geocoding via OpenStreetMap Nominatim
- Real-time route calculation using Leaflet Routing Machine
- Current location detection via browser Geolocation API
- Clear and user-friendly UI controls

## Technologies Used
- HTML5, CSS3, JavaScript
- Leaflet.js
- Leaflet Routing Machine
- OpenStreetMap (Nominatim)
- CARTO Basemaps
